- [Narrator] Add subscribers in bulk.
Bulk add allows you to assign
multiple subscriptions at once.
To get started, select Add
and choose Bulk Add from the dropdown.
Download the Excel template
from the dialogue box
and save it locally.
In the required fields of the template,
enter the details for the new subscribers.
Save the template again,
then return to the dialogue
box to upload your saved file.
Click OK and you'll see the newly
assigned subscriptions listed
in the grid below.
For more about Visual Studio
Subscription Assignments,
check out aka.ms/vss/admin/assignbulk
or some of our recommended
content in the links below.
